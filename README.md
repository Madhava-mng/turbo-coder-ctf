# turbo-coder-ctf

fast gui encoder and decoder tool for ctf

* Need: gui interface
* module: codelib
* package: python3-tk

**Installation:**
```bash
$ git clone https://github.com/Madhava-mng/turbo-coder-ctf.git
```

**aditional package:**

```python
$ python3 -m pip install codelib

$ apt install python3-tk
```
